qml-oscilloscope

Desarrollo de un osciloscopio casero de un canal que mide señales en una frecuencia de hasta 100 KHz en un rango de 0 a 3.3V.

Frameworks:
-Qt 5.8 (interfaz visual para PC)
-Thonny (captura de la señal y envío del Raspberry Pi Pico a la PC)

La versión actual es compatible solo con sistema operativo Windows.
